HOW TO INSTALL PUFFERPANEL ON GITHUB 1. create pufferpanel w/ commands down below 2. add account 3. Press restart server 4. Open port 5. done! $ apt update $ mkdir -p /var/lib/pufferpanel $ docker volume create pufferpanel-config $ docker create --name pufferpanel -p 8080:8080 -p 5657:5657 -v pufferpanel-config:/etc/pufferpanel -v /var/lib/pufferpanel:/var/lib/pufferpanel -v /var/run/docker.sock:/var/run/docker.sock --restart=on-failure pufferpanel/pufferpanel:latest $ docker start pufferpanel $ docker exec -it pufferpanel /pufferpanel/pufferpanel user add
